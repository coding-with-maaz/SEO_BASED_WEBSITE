## HTML Tags You Can Use

You can use these HTML tags in the article content:

- **Text Formatting:**
  - `<h1>`, `<h2>`, `<h3>`, `<h4>`, `<h5>`, `<h6>` - Headings
  - `<p>` - Paragraphs
  - `<strong>` or `<b>` - Bold text
  - `<em>` or `<i>` - Italic text
  - `<u>` - Underlined text
  - `<br>` - Line break

- **Lists:**
  - `<ul>` - Unordered list
  - `<ol>` - Ordered list
  - `<li>` - List item

- **Media:**
  - `<img>` - Images
  - `<video>` - Videos
  - `<iframe>` - Embedded content

- **Links:**
  - `<a href="...">` - Links

- **Structure:**
  - `<div>` - Container
  - `<span>` - Inline container
  - `<section>` - Section

- **Styling:**
  - Use `style` attribute for inline CSS
  - Use `class` attribute if your site has CSS classes

## Tips

1. **Always use inline styles** - Since this is rendered directly, inline styles ensure consistent appearance
2. **Use responsive images** - Set `max-width: 100%` and `height: auto` for images
3. **Add alt text** - Always include `alt` attributes for images
4. **Test your HTML** - Preview the content before publishing
5. **Keep it simple** - Avoid complex JavaScript or external stylesheets
